# Theme review framework with Behat

## Getting Started

```
$ npm install
$ npm run init
```

## Run Tests

Start PhantomJS.

```
$ npm start
```

Run tests.

```
$ npm test
```

Stop PhantomJS.

```
$ npm stop
```

Print all available step definitions.

```
$ bin/behat -di --lang=en
```
